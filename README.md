task-react-sample
=======

## Installation

```shell
$ git clone git@github.com:teloo-sandbox/task-react-sample.git
$ npm install
$ npm run build
```

## Usage

```shell
$ npm start
```

Access http://localhost:3000 on the browser.

## Test

```shell
$ npm test
```
